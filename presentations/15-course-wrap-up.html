Module 15

Course Wrap-up

What you built and where to go next.

What You Built

  • Static frontend on S3
  • REST API in API Gateway with Cognito authorizer
  • Lambda functions wired to DynamoDB and S3
  • Authenticated users via Cognito, per-user data isolation
  • Whole stack in AWS SAM, observable through CloudWatch + X-Ray

Habits to Keep

  • Least privilege every time — boring policies are good policies
  • Define access patterns first, schema second
  • Everything in IaC — the console is for reading, not writing
  • Log structured JSON with correlation IDs
  • Small, frequent deploys — the fastest way to stay calm

Where to Go Next

  • AWS Certified Developer – Associate (DVA-C02) — this course lines up with the exam
  • Serverless Land — pattern catalog at serverlessland.com
  • AWS Lambda Powertools for Python — production utilities (structured logging, tracing, idempotency)
  • AWS CDK — infra in your language; compiles to CloudFormation
  • AWS Well-Architected — run the review on anything you ship
