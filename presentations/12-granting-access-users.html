Module 12

Granting Access to Your Application Users

Amazon Cognito — sign-up, sign-in, JWTs.

Don't Roll Your Own

  • Password hashing, MFA, password reset, email verification — all easy to subtly break
  • Account recovery and lockout are regulatory minefields
  • Social IdPs need constant protocol updates
  • Use a managed service. Cognito, Auth0, Okta — pick one.

Cognito — Two Pieces

  • User Pool — directory; sign-up, sign-in, MFA, JWT issuance
  • Identity Pool — exchanges an ID token for temp AWS credentials (browser → AWS direct)
  • Most apps need only a user pool

What a JWT Looks Like

eyJhbGciOiJSUzI1NiIs...   header
.
eyJzdWIiOiJ1c2VyMSIs...   payload  (sub, email, exp, aud, iss)
.
MXp1ZVZ0ZGN...            signature
  • Three Base64-encoded sections joined by dots
  • Payload is readable — never put secrets in a JWT
  • Signature verifies the token wasn't tampered with

Tokens Cognito Issues

  • ID token — identity claims (sub, email, custom attrs)
  • Access token — scoped permissions; what your API trusts
  • Refresh token — exchanged for new access/ID without re-login
  • Default lifetimes: ID/access 1 h, refresh 30 d (configurable per app client)

OAuth Flow to Use

  • Authorization Code + PKCE — the right flow for SPAs and mobile
  • Client Credentials — machine-to-machine; no user
  • Avoid Implicit (deprecated) and Resource-Owner-Password (legacy)
  • Hosted UI gives you sign-up / sign-in pages out of the box

Protecting API Gateway

  • Attach a Cognito User Pool authorizer to each method
  • API GW validates signature, expiry, audience
  • Claims forwarded to Lambda via event.requestContext.authorizer.claims
  • Unauthenticated → 401; unauthorized → 403

Best Practices

  • Authorization Code + PKCE for browser apps
  • Never store tokens in localStorage — in-memory or http-only cookie
  • MFA on for privileged accounts; optional for the rest
  • Turn on advanced security for compromised-credential detection
  • Use Lambda triggers sparingly — each is a failure mode in your sign-in path
